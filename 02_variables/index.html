Variables in JavaScript

A JavaScript variable is just a name of storage location.

And we can give that name in three ways.

  • Using let keyword.
  • Using const keyword.
  • Using var keyword.

Rules for naming variables.

A variable name can conṭain any of the following charachters but must not start with a digit.

  • Capital Letters: A to Z

  • Small Letters: A to Z

  • Digit:0 to 9(but we cant't start variable name with digit)

  • Only two special charachters that are: Dollar"$" and Underscore"
